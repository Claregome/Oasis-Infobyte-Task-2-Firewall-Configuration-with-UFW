## Objective
The objective of this task was to configure a basic firewall using UFW (Uncomplicated Firewall) on a Linux system to control incoming network traffic.

## Tool Used
- UFW (Uncomplicated Firewall)

## System Used
- Ubuntu Linux

## Configuration Steps
1. Installed UFW on the system
2. Allowed SSH traffic to maintain secure remote access
3. Denied HTTP traffic to reduce the attack surface
4. Enabled the firewall
5. Verified firewall rules using status checks

## Firewall Rules Applied
- **SSH (Port 22):** Allowed
- **HTTP (Port 80):** Denied

## Conclusion
This task demonstrates basic firewall configuration and highlights the importance of controlling network access to improve system security.
